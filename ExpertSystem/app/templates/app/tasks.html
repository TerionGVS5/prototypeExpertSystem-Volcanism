Краткое руководство

Для решения задачи кластеризации предлагается два метода решения: линейного графа, метод масок. Непосредственное решение задачи предполагает выбор признаков, по которым выполняется проверка сходства/различия объектов (вулканов), выбранных на втором этапе определения исходных данных решения одним из методов. На третьем этапе необходимо выбрать вулканы для решения задачи кластеризации. Также для метода односвязного графа потребуется указать количество кластеров.
